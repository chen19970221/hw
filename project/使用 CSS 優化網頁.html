下列的 emmet 語法，哪一項是錯的？

  • ! = HTML5 環境
  • link:src = 匯入CSS
  • p*4 = 顯示4個p段落
  • h1 = 展開 h1 的開頭與結尾 html 標籤
  • lorem = 自動就幫我們寫完網頁
